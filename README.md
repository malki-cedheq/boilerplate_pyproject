# Boilerplate para gerenciamento de dependências com Poetry

## Configuração inicial
`poetry config virtualenvs.in-project true`

## Instalando dependências
`poetry lock`
`poetry install`

## Entrando no shell do poetry
`poetry shell`

## Tarefas

### linter
`task lint`

### formatter
`task format`

### executa aplicação
`task run_dev`
